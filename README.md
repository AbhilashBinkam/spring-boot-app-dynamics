# VM Args to configure AppDynamics Agent

```bash
-javaagent:C:\AppServerAgent\javaagent.jar
-Dappagent.install.dir=C:\AppServerAgent
-Dappdynamics.agent.applicationName=spring-boot-appdynamics-sample
-Dappdynamics.agent.tierName=local-tier
-Dappdynamics.agent.nodeName=local-node
-Dappdynamics.controller.ssl.enabled=true
-Dappdynamics.controller.hostName=bear202410120252164.saas.appdynamics.com
-Dappdynamics.controller.port=443
-Dappdynamics.agent.accountName=bear202410120252164
-Dappdynamics.agent.accountAccessKey=y084tboaccxs
-Dappdynamics.agent.loglevel=debug
```
